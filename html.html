HTML Skeleton

The HTML Skeleton is the basic structure of every HTML document. The HTML skeleton consists of the document type, html tags, a head, and a body.

2-Sided Tags

Two sided tags have an open and close tag. The close tag has a / before the element. You can code tags inside other tags. Examples of 2 sided tags are the p or paragraph tags and the em or italics tags.

1-Sided Tags

One sided tags do not have a closing tag. You can include the / at the end of the tag. One sided tags include br or break and hr or horizontal, br is used to end a certain part of the line off and hr is used to separate lines horizontally using a line.

Attributes

Attributes provide more info about a tag. They have the attribute name followed by an equal sign and a value in quotes.
